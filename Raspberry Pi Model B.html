Raspberry Pi Model B

Source:Efa, Location of connectors and main ICs on Raspberry Pi Model B revision 1.2, en.wikipedia.org (https://en.wikipedia.org/wiki/Raspberry_Pi#/media/File:Drawing_of_Raspberry_Pi_model_B_rev2.svg)

The Raspberry Pi Model B is a remarkably cheap computer, making it much more accessible than most competitors’ products. It’s capable of word-processing, running games, creating spreadsheets, plays HD video, and can run several types of Linux. Its small size is enabled by the Broadcom BCM2835, which carries an ARM1176JZFS, allowing the whole system to run at an impressive 700 mhz. Additionally, the VIdeocore 4 graphics card enables 30 frames per second 1080p video, equivalent to BluRay quality. Additionally, the Model B has an ethernet port which enables it to traverse the web. The two USB ports on the board also enable you do so with a mouse and keyboard, a much needed utility. This is all powered by simply plugging the provided power cord into the correct port, after which the Pi will turn right on. There is, however, no power button, so you will have to unplug the board to power it down. The pi was revolutionary for the time, providing all these utilities for a measly 35 dollars. Thus, it was able to supply schools and other groups with computers when they would have otherwise been required to stick with older methods of education. Besides technical information, the board also came with a guide on starting to use it, a booklet about the Raspberry Pi Foundation, a tutorial on wireless internet connection, a guide to create a Boot Card, and a tutorial to run DOOM, a classic video game, on the system. The Raspberry Pi Model B was a game-changer in the industry of cheap computers.
