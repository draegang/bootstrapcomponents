Raspberry Pi 3 Model B

Source:Efa, Location of connectors and main ICs on Raspberry Pi 3, en.wikipedia.org (https://en.wikipedia.org/wiki/Raspberry_Pi#/media/File:RaspberryPi_3B.svg)

For the same price as all of is ancestors, the Raspberry Pi 3 Model B introduced integrated wireless internet and general performance improvements. Unlike most new releases from the Raspberry Pi Foundation, the Raspberry Pi 3 Model B could not be summed up as a simply moderate efficiency increase or by a part swap or two. This was more than that. With the introduction of this model, wireless internet, Wi-Fi, was introduced to the realm of the mystical $35 computer. Where before you were required to have an ethernet cord stuck into your machine, the Raspberry Pi 3 Model B gave you the opportunity to use it like a laptop. Now, you could use the internet on your dirt-cheap board wirelessly. This was, obviously, a game0changer. Besides this massive change, the system is mostly the same. Still the same ports, similar power requirement, microSD, 40 GPIO pins, the standard fittings for any board before and during the era of the Raspberry Pi 3 Model B. This model also utilized Bluetooth to increase the amount of ways one could connect and utilize the internet. Compared to the previous Model B, this board ran about a third faster when doing most things due to a 1.2 GHz quad-core processor. Unfortunately for a majority of PC users at the time of the Raspberry Pi 3 Model B’s release, it was limited to Linux, like the Pi 2 before it. Windows eventually released their own version, as they did with the Pi 2, but it wasn’t exactly the version of Windows 10 users were used to.
